## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

## Dependencias/Comandos
Prisma Client
npm i @prisma/client

Dependencias de Desarrollo
npm i -D prisma 

Inicializar Prisma
npx prisma init
